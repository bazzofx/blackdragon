Vulnerability Findings

Slowloris DOS attack
LIKELY VULNERABLE CVE:CVE-2007-6750

Slowloris tries to keep many connections to the target web server open and hold them open as long as possible. It accomplishes this by opening connections to the target web server and sending a partial request. By doing so, it starves the http server's resources causing Denial Of Service.

References:
http://ha.ckers.org/slowloris/
https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2007-6750
HTTP Strict Transport Security (HSTS) Not Configured
MISSING

HSTS is not enabled on this HTTPS port. Without HSTS, browsers may connect over unencrypted HTTP, exposing users to man-in-the-middle downgrade attacks.

References:
https://cheatsheetseries.owasp.org/cheatsheets/HTTP_Strict_Transport_Security_Cheat_Sheet.html
HSTS Missing on HTTPS Ports

HTTP Strict Transport Security (HSTS) is not configured on the following HTTPS-enabled ports: 443, 2053, 2083, 2087, 2096. Without HSTS, users are vulnerable to SSL stripping and man-in-the-middle downgrade attacks.

Recommendation: Add the Strict-Transport-Security: max-age=31536000; includeSubDomains; preload header to all HTTPS responses.
