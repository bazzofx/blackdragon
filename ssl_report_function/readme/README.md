# 🛡️ TLS Report Generator (`generateTLSReport.py`)

The `generateTLSReport.py` script parses a raw scanner HTML report from `testssl.sh`, runs security/health scoring calculations, merges optional HTTP header audits from `curlReport.json`, and outputs a beautiful, client-ready, and enhanced HTML summary dashboard.

---

## 🚀 Usage

Run the script from the command line using python:

```bash
python generateTLSReport.py <input_html_file> [options]
```

### Options

| Flag | Argument | Description |
|---|---|---|
| `<input_html_file>` | File Path (Required) | The path to the raw input HTML report generated by the scanner. |
| `-o`, `--output` | File Path (Optional) | The output path for the enhanced HTML report. Default is `enhanced_report.html`. |
| `-locked` | None (Optional) | Generates a password-protected gated report with limited preview. |

---

## 🔒 Gated / Locked Feature (`-locked`)

When the `-locked` flag is supplied:
1. **Limited Preview**:
   - The **Vulnerabilities** tab shows only the first **3** findings.
   - The **Action Roadmap** tab shows only the first **3** action items.
   - All subsequent findings and items in those tabs are blurred under a password-protection overlay.
2. **Visual Cues**:
   - A padlock emoji (`🔒`) and a `locked-tab` class are automatically appended to the locked tab buttons.
   - A "Premium Content Locked" banner is placed at the top of the tab contents.
3. **Password Unlock**:
   - Entering the correct password (**`cybersamurai2024`**) into any overlay unlocks that tab, immediately unblurring and revealing all hidden findings.

---

## 💡 Examples

### 1. Generate a Full Report (Un-gated)
To generate a standard, fully visible report containing all parsed findings:
```bash
python generateTLSReport.py test_units/rawTLSReport.html -o enhanced_report.html
```

### 2. Generate a Locked / Gated Report
To generate a password-protected report where findings beyond the first 3 are locked:
```bash
python generateTLSReport.py test_units/rawTLSReport.html -o enhanced_locked_report.html -locked
```
